Project Description

Project Proposal Description

Perturbations can be introduced into gene expression data in a variety of ways - some natural (e.g. minor expression differences due to individual genomic events) and some artificial (e.g. instrumentation error in measuring). In this study, a method for analyzing the ability for genetic classifiers, in the presence of perturbed data, to resist misclassification (numerical stability) will be developed.

Three classifiers will be analyzed: kNN, linear SVM and NBC. Briefly, kNN classifies samples based on similarity scores of its k nearest neighbors. Linear SVM is a binary classifier which can be adapted for multi-class datsets. It separates the given data points into their correct classes with a constructed hyperplane with the maximum margin that separates the two classes. NBC is a correlation-based classification algorithm that constructs a function to determine a theoretical expression for each sample for each class, assigning the sample to the class with the closest matching expression.

The developed method will use a divide and conquer approach, along with a genetic algorithm (GA) to determine scores for each classifier, allowing them to be ranked from most robust to least. The method will then be compared to other techniques.

Implementation

First, I will implement the kNN, SVM and NBC classifiers. Second, I will implement Chi2 and random feature selection. Third, I will implement the stability analysis method according to the following:

Given a user specified robustness and a number of classifiers, the method will order the classifiers from most robust to least. Here robustness refers to the missclassification rate.

A divide and conquer method will be used to select the number of features (k) to be altered in a test sample. For each selected k, a genetic algorithm will be used to select those k features (genes) which give the greatest missclassification rate. This method is repeated until k is minimized for a user defined robustness. The algorithms will be ranked according to k, with smaller k being ranked higher.

All implementation will be completed using a mix of Python and C++.

Methods

Linear SVM and kNN are readily available for use in packages. NBC can be found described in the original paper and will be implemented. Chi2 is available in a package, while the random and GA will be implemented.

Data

The following datasets will be used to analyze the method: Lung Cancer, Breast Cancer, Colon Cancer

Measurements

The GA feature selection algorithm will be compared to 10-fold random selection and Chi2 feature selection, with robustness score for a given k being used to compare them. Classifiers will be ranked according to k, as described previously.
