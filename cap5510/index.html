CAP5510 | Project Proposal

A method for analyzing numerical stability of genetic classifiers.

  • GitHub Repo
  • Results Page
Intoduction

Perturbations can be introduced into gene expression data in a variety of ways - some natural (e.g. minor expression differences due to individual genomic events) and some artificial (e.g. instrumentation error). In this study, a method for analyzing the numerical stability of genetic classifiers will be developed. The developed method will use a divide and conquer approach, along with a genetic algorithm (GA) to determine scores for each classifier, allowing them to be ranked from most to least robust. Robustness refers to the missclassification rate, with more robust algorithms having a smaller misclassificatin rate.

Background

Three classifiers will be analyzed: kNN, linear SVM and NBC. Briefly, kNN classifies samples based on similarity scores of its k-nearest neighbors. Linear SVM is a binary classifier which can be adapted for multi-class datsets. It separates the given data points into their correct classes by constructing a hyperplane that separates the classes with the maximum margin. NBC is a correlation-based classifier that constructs a theoretical expression for each sample of each class, assigning the sample to the class with the closest matching expression.

Methods

First, the kNN, SVM and NBC classifiers will be implemented. Linear SVM and kNN are readily available for use in packages. NBC can be found described in the original paper and will be implemented. Second, Chi2, and random feature selection will be implemented.Chi2 is available in a package, while the random method will be implemented. Third, I will implement the stability analysis method according to the following rules.

Given a user specified robustness and a number of classifiers, the method will order the classifiers from most to least robust. A divide and conquer method will be used to select the number of features (k) to be altered in a test sample. For each selected k, a genetic algorithm will be used to select k features (genes) which give the greatest missclassification rate. This method will be repeated until k is minimized for the user defined robustness.

The algorithms will be ranked according to k, with smaller k being ranked higher. Additionally, the GA algorithm will be compared to 10-fold random selection and Chi2 feature selection, with robustness scores for a given k being used to compare them. All implementation will be completed using a mix of Python and C++.

Data

The following datasets will be used in the analysis: Lung Cancer, Breast Cancer, Colon Cancer.
