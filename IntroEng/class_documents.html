Class Documents

Missed Something? Want to Review? We've got you covered.

In class, we go over a lot of information, including details about the CISE Department, research offerings, and overall job outlook for graduating computer scientists and engineers. We even have a bit of fun coding. Hopefully the information is both enlightening and useful. Below are materials presented in class, as well as a few extras. Just click the download button next to the description to get the corresponding files. For more information on how to run the files and the full repository, please visit the GitHub Repo. We hope to see you around the CISE Department in the near future and wish you success in all your endeavors.

Type Description Download
.pdf A PDF version of the slides presented in class.
.sh The shell script written and demonstrated in class. Runs on LINUX.
.bat A batch file similar to what what was written in class. Runs on Windows.
.scpt An AppleScript similar to what was written in class. Runs on MacOS.
.app An application of the script code above. Runs on MacOS.

Note: For further instructions on each file and how to run the code, please visit the GitHub repository.
